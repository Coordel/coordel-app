Current
Current tasks are those from all of the users Coords that can be worked on now. This is a critical feature. We want to avoid the "Inbox" imagery because there is the invitation process that handles incoming tasks. This needs to indicate that work can be done on the task.
Deferred
Deferred means that a Task won't start until some time in the future.
Delegated
Delegated shows all tasks that are Delegated to someone else but that aren't part of any Coord.
Blocked
Blocked shows all Tasks that can't be worked on because another task isn't done that must be completed first.
Private
Private Tasks are those that I'm doing myself so don't involve anyone else.

Coord Responsible
If the user is responsible for a Coord, it looks like this. The Coord is something that will be branded so this is an important icon. It should avoid using project imagery of any kind.

The star is the element that indicates responsibility in the current application but can be changed.
Coord Responsible with Invitations
This is the icon that shows when there are invitations pending in a responsible's project
Coord Responsible with Flags
This is the icon that shows when invitations are declined, when someone leaves a project, etc in a responsible's project
Coord Participant
This is the icon for Coords in which the user is a Participant
Coord Pending
This is the icon for Coords that the responsible has created but hasn't yet sent to participants and followers
Coord Following
This is the icon for Coords that the user is Following
